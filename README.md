# Cricket Data Analytics using Python (Pandas) & Power BI

## Overview
This project focuses on analyzing cricket match and player performance data using Python (Pandas) 
for data cleaning and analysis, followed by Power BI for visualization and insights.

## Tools & Technologies
- Python (Pandas, NumPy)
- Jupyter Notebook
- Power BI
- CSV datasets

## Data Workflow
1. Loaded cricket datasets into Python
2. Cleaned and transformed data using Pandas
3. Performed exploratory data analysis (EDA)
4. Created interactive Power BI dashboards

## Key Insights
- Evaluated player and team performance
- Identified top-performing players
- Analyzed match trends and statistics
